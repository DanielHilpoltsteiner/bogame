# bogame
OGame bot written for Python3.

## Requirements

```
pip install -r requirements.txt
```

[protoc 3.5.1](https://github.com/google/protobuf/tree/master/python)

## Usage

```
protoc */*.proto --python_out=.
python main.py [-i=<PLAYER_FILE>]
```
